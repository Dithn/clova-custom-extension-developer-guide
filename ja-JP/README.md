# はじめに
[Clova](https://clova.line.me/)は、{{ book.TargetServiceForClientAuth }}が開発およびサービスを提供しているAIプラットフォームです。サードパーティの開発者は、Clovaの持つ技術を活用して、AIサービスを提供するデバイスまたは家電製品を製作できます。また、Clovaを利用して、保有しているコンテンツやサービスを提供することもできます。
